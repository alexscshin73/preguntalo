아직 등록된 매뉴얼이 없습니다. 오른쪽 1단계에서 첫 매뉴얼을 추가하세요.
1

지식 등록

서비스나 도메인 단위로 매뉴얼을 등록해 지식 베이스의 기본 단위를 만듭니다.

2

RAG 입력 파일 업로드

업로드한 파일은 파싱, 섹션 분리, 청킹을 거쳐 검색 가능한 RAG 데이터로 변환됩니다.

업로드 가능한 형식: PDF, DOCX, TXT, MD. 업로드가 끝나면 해당 버전이 즉시 검색 후보로 반영됩니다.
3

인덱싱 현황

선택한 매뉴얼의 버전별 상태를 보고, 질문 화면에 투입할 준비가 되었는지 확인합니다.

왼쪽 목록에서 매뉴얼을 선택하면 버전 현황이 표시됩니다.
